## Installation

```sh
npx degit shinokada/flowbite-svelte-starter my-demo
cd my-demo
pnpm i // or npm i
pnpm run dev // or npm run dev
```

Then update dependencies:

```
pnpm update
// you may need to run this:
pnpm i -D flowbite-svelte@latest
```
